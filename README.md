# homebrew-tap

Homebrew tap for [marka.md](https://markamd.vercel.app/).

## install

```sh
brew tap mattenarle10/tap
brew install --cask marka-md
```

Or install without tapping first:

```sh
brew install --cask mattenarle10/tap/marka-md
```

## update

marka.md has its own signed in-app updater. Homebrew users can also upgrade through brew after this tap is bumped for a new release:

```sh
brew update
brew upgrade --cask marka-md
```
